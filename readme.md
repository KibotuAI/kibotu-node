Kibotu-node
=============

Installation
------------

    npm install kibotu

Quick Start
-----------

```
// grab the Kibotu factory
var Kibotu = require('kibotu');

// create an instance of the kibotu client
var kibotu = Kibotu.init('<YOUR_TOKEN>');

```



Tests
-----

    # in the kibotu directory
    npm install
    npm test
